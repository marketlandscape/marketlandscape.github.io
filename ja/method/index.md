# デザインと考え方

ナビゲーション・インデックスは、当初からトレーディングシグナルとして設計されたものではなく、価格を予測するためのモデルでもありません。その役割は別にあります。暗号資産市場全体の状態を相対的なポジショニングの尺度に変換し、市場サイクルの中で現在どの位置にあるのかをより把握しやすくすることです。

この構成は、いくつかの重要な考え方に基づいています。

第一に、市場サイクルの段階は、短期的な価格変動よりも重要です。暗号資産市場においては、日々のボラティリティや局所的な値動きよりも、サイクルの構造や現在の市場環境のほうが大きな意味を持つことが多くあります。

第二に、過去の価格推移には有用な情報が含まれていますが、それだけで全体像が得られるわけではありません。過去データは、将来を当てるための手段ではなく、現在の市場状態を過去に観測された局面と照らし合わせるための経験的な基準として用いられます。

第三に、想定される価格レンジは意図的に限定されており、投機的な性格を持たないよう設計されています。予測的な要素は極めて保守的に設計されており、ナラティブやノイズ、過度な市場の熱狂がインデックスの値に与える影響を抑えることを目的としています。

これらのインデックスは、市場サイクルの中での位置や大まかな方向性を理解する助けにはなりますが、正確な売買のタイミングを示すものではありません。設計の中心にあるのは、短期的な価格の動きへの反応ではなく、安定性です。値はゆっくりと変化し、あらかじめ定められた長期的な考え方との整合性を保ちながら、恐怖や過度な楽観といった感情の影響を抑えることを目的としています。
